Реализовано:
- поддержка аудио
- продвинутый алгоритм для сравнения совпадения двух строк. Строки являются равными, если расстояние Левенштейна достаточно мало.
- три режима:
  - single choice
  - multi choice
  - typing
- смена сложности
- криповая infinite-cycle музыка
- шорткат для помощи (H)
